## Confusion matrix
1. True positive
2. True negative
3. False positive
4. False negative

## Sensitivity
TP / (TP + FN) : 有多少比例的實際 positive 的例子會被找出來。
追求 model 的 sensitivity 表示找出 P 比找出 N 重要。

## Specificity
TN / (TN + FP) : 有多少比例的實際 negative 的例子會被找出來。
追求 model 的 specificity 表示找出 N 比找出 P 重要。
